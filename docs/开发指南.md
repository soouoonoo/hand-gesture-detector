# 开发指南（初期版）

## 开发环境配置

### 1. 安装Python
- 下载Python 3.9+：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### 2. 安装Git
- 下载Git：https://git-scm.com/downloads
- 全部使用默认选项安装

### 3. 获取项目代码
```bash
# 克隆项目到桌面
cd Desktop
git clone https://github.com/your-team/hand-gesture-detector.git
cd hand-gesture-detector
```

### 4. 创建虚拟环境

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate.bat
# Mac/Linux:
source venv/bin/activate
```

### 5. 安装依赖包

```bash
pip install -r requirements.txt
```

### 6. 测试安装

```bash
python tests/test_camera.py
```

## Git使用指南

每日工作流程

```bash
# 早上开始工作
git pull origin main          # 获取最新代码
git checkout -b feature/你的功能  # 创建功能分支

# 开发过程中
git add .                     # 添加修改
git commit -m "描述你的修改"    # 提交

# 晚上结束前
git push origin feature/你的功能  # 推送到远程
# 然后在GitHub创建Pull Request
```

## 项目结构说明

src/ 源代码目录

· main.py: 主程序入口
· hand_detector.py: 手部检测器
· utils.py: 工具函数

tests/ 测试目录

· test_camera.py: 摄像头和环境测试

docs/ 文档目录

· 项目文档和指南

## 常见问题

### Q: 摄像头打不开

1. 检查摄像头是否正确连接
2. 关闭可能占用摄像头的软件（微信、QQ等）
3. 修改摄像头索引（0改为1或其他）

### Q: pip安装失败

```bash
# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: Git推送失败

1. 先执行 git pull origin main
2. 解决可能的冲突
3. 再执行 git push

## **立即开始使用的步骤**

### **步骤1：创建项目结构**
```bash
# 1. 创建项目文件夹
mkdir hand-gesture-detector
cd hand-gesture-detector

# 2. 创建所有目录
mkdir -p src tests docs

# 3. 创建上述所有文件
# （复制上面的内容到对应文件）

# 4. 初始化Git
git init
git add .
git commit -m "初始化项目结构"
```

步骤2：测试运行

```bash
# 1. 安装依赖
pip install opencv-python mediapipe numpy

# 2. 运行测试
python tests/test_camera.py

# 3. 运行主程序
python src/main.py
```

步骤3：开始开发

1. 每个成员克隆仓库
2. 按照开发指南配置环境
3. 从简单的任务开始：
   · 修改界面显示文字
   · 调整检测参数
   · 添加新的显示信息
