# 手势识别项目

## 项目简介
基于OpenCV和MediaPipe的实时手势识别系统。

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

## 运行测试

```bash
python tests/test_camera.py
```
## 运行主程序

```bash
python src/main.py
```
## 项目结构

```bash
hand-gesture-detector/
├── src/          # 源代码
├── tests/        # 测试
└── docs/         # 文档
```
